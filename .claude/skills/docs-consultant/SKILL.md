# Living Docs Consultant

## When to Use
- Before taking on tasks that depend on portfolio priorities or domain context.
- When an agent needs a quick synopsis of Level 0/1/2 docs without manually opening each file.

## Workflow
1. Read `docs/00-portfolio-digest.md` for current heartbeat and blockers.
2. Load relevant files under `docs/domains/` and `knowledge/domains/` for domain specifics.
3. Check `docs/20-reference-index.md` and `docs/22-root-data-index.md` for supporting material.
4. Provide a concise summary: active focus, key risks, next steps, and links.
5. Flag any stale artefacts that need updating.

## Tips
- Keep summaries to actionable bullet points.
- Note dates/last updated fields so follow-up agents know freshness.
- If discrepancies appear between knowledge repo and project docs, raise them (or pass to the updater skill).
