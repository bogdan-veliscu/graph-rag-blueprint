## Temporary Files

Temporary analysis, implementation notes, status reports, and test results have been moved to `scratchpad/docs_archive/` per `AGENTS.md` guidelines.

If you need to reference temporary documentation:
- Check `scratchpad/docs_archive/` for historical analysis and implementation notes
- These files can be safely deleted after epics are completed

## Adding New Documentation

When adding new documentation:
- **Permanent docs** → Place in `docs/`
- **Temporary analysis/notes** → Place in `scratchpad/`
- See `AGENTS.md` for detailed guidelines
